GNM Code

Description: 

The code takes a file containing the results from EvFold
It uses the sequence information to create a GNM and obtain dynamics

Usage:

gnm.py FILE

Output:

A new folder will be created for a given PDB code containing: 
    - bfactors.txt
    - sqflucts.txt
    - kirchhoff.txt
    - figure of bfactors and correlations
